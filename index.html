Kişisel Web Sayfama Hoş Geldiniz!

Merhaba! Ben Yusuf Gökalp Kutlu, web sayfama hoş geldiniz. Fırat Üniversitesi Adli Bilişim Mühendisliği 2. sınıf öğrencisiyim ve özellikle Python ve Flask ile web geliştirme alanında kendimi geliştirmeye çalışıyorum. Bu web sayfamda, daha önceki projelerimden bazı örnekler bulabilirsiniz. Python ve Flask kullanarak oluşturduğum web uygulamaları, RESTful API'lar ve daha fazlası gibi teknik becerilerim hakkında bilgi edinebilirsiniz. Benimle çalışmak isterseniz, lütfen iletişim sayfamı ziyaret edin.

Ayrıca bu yaz, okulumun zorunlu staj programına katılacağım. Okulda aldığım dersler ve dışarıda edindiğim deneyimler sayesinde, backend geliştirme, web uygulamaları, ve veritabanı tasarımı gibi konularda da bilgi sahibiyim. Bu konulardaki bilgilerimi staj sürecinde kullanmak istiyorum. Eğer sizin şirketinizde veya tanıdıklarınızın çalıştığı bir yerde staj yapma fırsatım olursa çok memnun olurum.
